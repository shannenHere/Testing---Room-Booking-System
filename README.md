Uses `jUnit4`.

---

### Local Setup & Installation
Prerequisties:  Java JDK 8 (or newer) & jar_files.  
<code>git clone https://github.com/shannenHere/shannenHere/Testing---Room-Booking-System.git</code> 

In eclipse:  
File > Import
General > Projects from Folder or Achive  
Directory > Testing---Room-Booking-System

---

### Launch System
Right click on HotelBookingSystem.Java > Run as > Java Application

### Launch Tests
Right click on individual test files (recommended) / test subfolder > Run as > jUnit Test
